Напиши сайт на html, css, JS и по возможности визуализируй результат
Платформа для объективного сравнения и выбора нейросетей (текстовых, графических, аудио/видео) в едином интерфейсе.

Стиль: Современный веб-дизайн (2025), белый фон, много воздуха, четкая сетка. Использовать элементы дашборда: графики, чекбоксы, переключатели, карточки с данными.

Блоки:

Хедер:
Логотип «NeuroCompare». Навигация: Каталог моделей, Сравнение, Цены, О проекте.

Хиро-секция:
Заголовок: «Объективное сравнение нейросетей для любых задач».
Подзаголовок: «Выбирайте идеальную модель по качеству, скорости и стоимости. Все популярные нейросети в одном месте».
Визуальная часть: интерактивный график «Соотношение цена/качество для топ‑10 моделей» и облако из логотипов нейросетей: GPT‑4, Claude, Midjourney, Stable Diffusion, DALL‑E, Gemini.

Категории моделей (табы):
Три вкладки: «Текстовые», «Графические», «Аудио/Видео».
Под активной вкладкой — сетка из 4 карточек популярных моделей данной категории. Каждая карточка содержит: название модели, краткое описание (ключевые особенности), рейтинг (звёзды), иконку модели.

Таблица сравнения:
Заголовок: «Сравните характеристики».
Таблица с фильтрами: выбор моделей для сравнения через чекбоксы.
Колонки: Модель, Контекст (токенов), Скорость (токенов/сек), Цена за 1K токенов, Качество генерации (оценка), Поддержка языков, Мультимодальность (да/нет).
Строки: GPT‑4, Claude 3 Opus, YandexGPT, Midjourney, DALL‑E 3.

Отзывы и кейсы:
Заголовок: «Как нейросети меняют рабочие процессы».
Две карточки с примерами от профессионалов:
«Сократили время создания иллюстраций на 60% благодаря подбору оптимальной графической модели» — дизайнер Анна.
«Ускорили анализ документов в 5 раз, выбрав GPT‑4 для обработки текста» — аналитик Михаил.
Каждая карточка содержит: имя, фото (иконка), цифру результата, логотипы использованных нейросетей.

CTA (призыв к действию):
Заголовок: «Начните тестировать нейросети прямо сейчас».
Подзаголовок: «Сравнивайте, выбирайте и интегрируйте лучшие модели для своих задач».
Крупная кнопка контрастного цвета (ярко‑синяя или фиолетовая) с текстом «Начать тестирование».
Под кнопкой мелким текстом: «Бесплатно, без регистрации, 3 тестовых запроса в день».

Детали оформления:

Везде округлые углы, тонкие тени, аккуратные линии.

Для дашборд-элементов — минималистичные иконки и графики: стилизованные заглушки (линия графика с точками).

Основная цветовая гамма: белый фон (#FFFFFF), тёмно‑серый текст (#1E1E1E), акцентный цвет — сине‑фиолетовый (#4A6CF7) для кнопок и выделений.

Шрифты: современный гротеск (Inter, SF Pro Display) — тонкое начертание для заголовков, обычное для текста.

Отступы между секциями: padding не менее 60–80px.
